What Is It?

WheresMyIp.org is an open-source and crowd-sourced-data project striving to provide free as-accurate-as-possible IP to geographic location information for the IoT age. As devices become more portable and location-aware, getting exact information for the location of any device (especially IPv6 devices) will get more and more difficult. Although client devices know where they are, there's no way for servers (or other clients in a Peer-To-Peer environment) to know where each-other are.

Although commercial databases (like MaxMind) exist, they cost a lot of money and can't keep up-to-date with edge devices. Thus, we're attempting to create a crowd-sourced database driven by the community

How do I contribute information?

Grab one of our client SDKs to start contributing data

Browser SDK:

<script src="https://cdn.jsdelivr.net/gh/issacg/wheresmyip/wheresmyip-browser.js"/></script>
<script>wheresmyip.submit();</script>

Do-it-yourself

Make a POST request to https://api.wheresmyip.org/ip
The POST content should be type application/json and contain a JSON object containing the two properties lat and lon with decimal lattitude/longitude coordinates.

For example:

{"lat": 41.9040, "lon": 12.4530}

How can I get data?

Send requests to https://api.wheresmyip.org/ip/<address>

where "address" is a valid IPv4 or IPv6 address. Bear in mind that as a young and crowdsourced project, there isn't much data yet...

The output will be a JSON object containing the following fields:

  • IP - The ip address requested
  • Lat - The last known lattitude associated with the IP
  • Lon- The last known longitude associated with the IP
  • LastUpdated - The unix-timestamp of the last update for the given IP

FAQ

  • Do you support IPv6?

    • Sortof. The database and application support it. It's currently not on a dual-stack endpoint, so can't accept IPv6 submissions, but once the endpoint is dual-stack, it will work.

  • Can I set it up on my own?

    • There will (hopefully soon) be a script to install the entire stack on your Amazon AWS account. If you don't get much traffic, you'll be fine on the free tier. Check the /install folder on GitHub

  • Can I contribute code

    • Fork us and submit pull requests

  • Why does it look like sh*t?
